Presentación y Estilo

Aunque el diseño se maneja con CSS, etiquetas como "img" o "video" permiten insertar elementos multimedia. HTML define la estructura, CSS la apariencia.




Función: Define cómo se ven los elementos HTML (colores, tamaños, posiciones).
CSS (Cascading Style Sheets): Es el lenguaje para estilizar HTML.

Cómo aplicarlo:

Interno (en head): Usando la etiqueta style para estilos que aplican a todo el documento.
Externo (recomendado): Vinculando un archivo .css externo con la etiqueta link dentro del head, lo que permite mantener estilos para todo un sitio web, como se describe en este video y este recurso.
En línea no recomendado para mucho estilo: Usando el atributo style="" directamente en una etiqueta HTML para un estilo específico, como en este ejemplo.
