Proyecto HTML – TechEdu Solutions
Este proyecto consiste en el desarrollo de una página web básica utilizando HTML. El objetivo es presentar información clara y organizada sobre TechEdu Solutions, una empresa dedicada a la educación tecnológica, robótica educativa y servicios innovadores.
Características

Estructura semántica en HTML.
Secciones informativas: Inicio, Acerca de Nosotros, Cursos, Contacto.
Diseño adaptable para diferentes dispositivos (responsivo).

Tecnologías utilizadas

HTML5 para la estructura del contenido.
CSS (opcional) para estilos y diseño visual.
JavaScript (opcional) para interactividad.

Cómo visualizar el proyecto

Descarga o clona el repositorio.
Abre el archivo index.html en tu navegador.
